# latin-corpus-tools

Processing tools for Latin text corpora and resources.

## `amiatinus_extract.py`

Clones the [`bedes-bible`](https://github.com/adunning/bedes-bible) TEI edition of the Codex Amiatinus pandect and converts it to JSON and plaintext formats. Requires `lxml` and `GitPython`.
